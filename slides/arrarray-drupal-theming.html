es el cambio ## Browser Support - ≥IE9 - SVG (no Android 2.3) ## HTML(5) - formato schema.org (RDFa) - Mejor accessibilidad. - Atributos WAI-ARIA. - Atributos _data_. - Niveles DOM: D7 17, D8 17 ## CSS(3) - Menos \#IDs - SMACSS & BEM - Clases definidos en TWIG - [d.o/coding-standards/css](https://www.drupal.org/coding-standards/css) ## Javascript - jQuery v2.x - Modernizr - detección - Backbone.js - MVC - Underscore.js - utilidades ## Templating - _PHPTemplate_ => _Twig_ - `theme_*` y `*.tpl.php` => `*.html.twig` - Mas limpio. - Mas seguro. - Mas facil. ## Performance - _Agregación de CSS y JavaScript_ por defecto. ## Components - Mejor experiencia admin - Nuevos elementos - Botones desplegables - Dialogos modales ## Responsivo - Themes, toolbar, imagenes, y tablas - Breakpoint media queries ![backgrounds](img/sunset.jpg "background") ## Más Allá ## SASS - Extender la sintaxis de CSS con: - `$variables` - reglas encajadas - inline `@imports` - `@mixins` (funciones) ## Compass - caja de herramientas ## Recursos - [caniuse.com](http://caniuse.com/svg) - [d.o/documentation/theme](https://www.drupal.org/documentation/theme) - [#2356951: Diferencias de Theming 6,7, y 8](https://www.drupal.org/node/2356951) #### Gracias!!
